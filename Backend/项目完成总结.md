## 🚀 立即开始使用

### 启动系统
```bash
npm run dev
```

### 访问页面
- 🏠 **主页导航**: http://localhost:3001/
- 📋 **问诊表单**: http://localhost:3001/web  
- 🔧 **API演示**: http://localhost:3001/demo
- 🧪 **系统测试**: http://localhost:3001/test

### 使用流程
1. **填写问诊信息** - 在 `/web` 页面填写患者信息
2. **保存到云端** - 点击"保存到云端数据库"按钮
3. **查看记录** - 使用数据库操作按钮查看已保存数据
4. **搜索患者** - 输入患者信息进行快速搜索
